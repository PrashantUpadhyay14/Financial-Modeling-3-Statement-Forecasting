# 📈 Financial Modeling | Three-Statement Financial Forecasting Model

## 📌 Project Overview

This project is a fully integrated three-statement financial model developed in Microsoft Excel. The model forecasts a company's financial performance over a five-year period by linking the Income Statement, Balance Sheet, and Cash Flow Statement with supporting schedules for Working Capital, Capital Expenditure (CapEx), and Debt.

The model demonstrates financial forecasting, statement integration, ratio analysis, and dynamic financial modeling techniques commonly used in Financial Planning & Analysis (FP&A), Investment Banking, Equity Research, and Corporate Finance.

---

# 🎯 Business Objective

The objective of this project is to build a dynamic financial model capable of forecasting a company's future financial position by integrating historical financial statements with key operating assumptions and supporting schedules.

- EBIT Margin

- Current Ratio

- Cash Ratio

## 9️⃣ Financial Statement Integration

The model links all supporting schedules into the core financial statements, ensuring automatic updates throughout the workbook.

Examples include:

- Net Income → Balance Sheet & Cash Flow
- Depreciation → Income Statement, PPE Schedule & Cash Flow
- Working Capital → Cash Flow Statement
- Debt Schedule → Interest Expense & Balance Sheet
